###Iniciando o Sistema

Instalação dos modulos pelo node.js

####Create-react-app

npm install create-react-app --dev 

Execução do projeto:

npm start

link: http://localhost:3000/

###SASS

npm install --dev sass

Execução do comando SASS:

sass --watch src/style/scss/App.scss src/style/css/App.css

ou

npm run style
